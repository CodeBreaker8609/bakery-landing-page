# Velvet Crumb Bakery

Velvet Crumb Bakery is a modern, elegant bakery website built with HTML, CSS, and JavaScript. It was designed as a polished front-end project for a cake-focused bakery brand, featuring responsive layout design, interactive navigation, and a custom cake inquiry form.

## Project Overview

This project was created to practice building a real-world business website from the ground up using core front-end technologies. The goal was to create a bakery landing page that feels refined, modern, and user-friendly while also demonstrating practical front-end development skills.

The website focuses on:

- strong visual hierarchy
- elegant branding
- responsive layout structure
- interactive JavaScript features
- clean section-based page design
- form validation and user feedback

## Features

- Sticky responsive navigation bar
- Mobile hamburger menu
- Active section highlighting while scrolling
- Smooth scrolling navigation
- Full-width hero section with background image overlay
- Featured cake card layout
- About section with bakery highlights
- Custom order process section
- Contact and cake inquiry form
- Form validation with error and success states
- Character counter for order details
- Responsive layout for tablet and mobile screens

## Built With

- HTML5
- CSS3
